API automation of https://jsonplaceholder.typicode.com/ using mocha framework
- Go to https://jsonplaceholder.typicode.com/
- Go to route section
 I got these routes:

GET
/posts

GET
/posts/1

GET
/posts/1/comments

POST
/posts

PUT
/posts/1

DELETE
/posts/1

GET : Retrieve data from API
POST : Create data from API
PUT : Update full data from API
DELETE : Delete data from API


Initialize project
npm init -y
Install this packages:
npm i axios mocha chai
